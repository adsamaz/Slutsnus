<task type="checkpoint:human-verify" gate="blocking">
  <what-built>
    Complete Snus Catcher core loop:
    - Server: Full physics engine (spawn, fall, collision, lives, win) running at 20Hz
    - Client: 60fps Canvas 2D game with client-authoritative bar rendering
    - Platform: Socket.IO state sync, leaderboard write on game end, return-to-lobby
  </what-built>
  <how-to-verify>
    Start the development server: `npm run dev` from the workspace root.

    **Step 1 — Room creation and game start**
    Open two browser tabs (or one regular + one incognito). Log in as two different users.
    Tab A: Create a new Snus Catcher room from the lobby.
    Tab B: Join the room using the room code.
    Both tabs: Click "Ready". Tab A (host): Click "Starta spel".
    Expected: Both tabs transition from the lobby to the Snus Catcher canvas.

    **Step 2 — Falling objects and bar control**
    Move the mouse over the canvas on Tab A.
    Expected:
    - The bar follows the cursor immediately with no visible lag (client-authoritative)
    - Green circles (fresh snus pouches) fall from the top of the canvas
    - Red circles (spent snus pouches) also appear periodically among the falling objects
    - Bar on Tab B shows Tab A player's bar position (slightly delayed — from server state)

    **Step 3 — Score increment**
    On Tab A: Position the bar under a falling green (fresh) pouch and let it fall onto the bar.
    Expected:
    - The score "Poäng: X" in the top-left of the canvas increments
    - Tab B shows Tab A's updated score in the opponent HUD line
    - Catching does not cost a life

    **Step 4 — Life decrement**
    On Tab A: Let a red (spent) pouch fall onto the bar deliberately.
    Expected:
    - The life indicator (3 green circles in the top-right) decrements: one circle turns gray
    - Lives shows 3 → 2 → 1 as you touch more spent pouches

    **Step 5 — Game end and end screen**
    Continue until one player reaches 0 lives (let Tab A or B lose all 3 lives).
    Expected:
    - Both tabs switch from the canvas to the end screen
    - End screen shows: "Du vann!" on the winner's tab, "[Username] vann!" on the loser's tab
    - A results table shows both players' ranks (1st / 2nd), usernames, and scores

    **Step 6 — Leaderboard write**
    After the game ends, navigate to the leaderboard page.
    Expected: A new entry appears for both players with the game type "snus-catcher" and their final scores.

    **Step 7 — Return to lobby**
    On either tab: Click "Tillbaka till lobbyn".
    Expected: Hard navigation to '/' — the lobby appears. The game room is gone.
  </how-to-verify>
  <resume-signal>Type "approved" if all 7 steps pass. If any step fails, describe the issue (e.g., "Step 3: score does not increment", "Step 5: end screen does not appear") and Claude will fix it.</resume-signal>
