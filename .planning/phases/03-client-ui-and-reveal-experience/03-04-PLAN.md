Helper to compute combo level for a card given an event:
```typescript
function comboLevel(card: SnuskingCardInstance, event: SnuskingEventCard | null): 'gold' | 'silver' | null {
  if (!event || !card.strength || !card.flavor) return null;
  const strMatch = event.strengthAffinity.includes(card.strength);
  const flvMatch = event.flavorAffinity.includes(card.flavor);
  if (strMatch && flvMatch) return 'gold';   // 2x
  if (strMatch || flvMatch) return 'silver'; // 1.5x
  return null;
}
```

Render:
```
<div class="reveal-overlay">
  <!-- Self cards row with flip animation -->
  <div class="reveal-row self-reveal">
    <For each={props.selfCards}>
      {(card) => (
        <div class="reveal-card-flip">
          <SnuskingCard
            card={card}
            comboLevel={comboLevel(card, props.currentEvent)}
          />
        </div>
      )}
    </For>
  </div>

  <!-- Opponent rows: face-down (they have committed but we don't see their cards) -->
  <For each={props.opponents}>
    {(opp) => (
      <div class="reveal-row opponent-reveal">
        <span class="reveal-player-label">{opp.username}</span>
        <For each={Array.from({ length: opp.handCount })}>
          {() => (
            <div class="reveal-card-flip">
              <SnuskingCard />
            </div>
          )}
        </For>
      </div>
    )}
  </For>
</div>
```

The `.reveal-card-flip` class (in snusking.css) runs the `card-flip` keyframe animation once on mount, achieving the simultaneous 3D flip effect. The animation is 350ms, `animation-fill-mode: forwards`, `transform-style: preserve-3d`.

**Event flash timer logic:**
```typescript
const [eventFlashActive, setEventFlashActive] = createSignal(false);
const [lastEventId, setLastEventId] = createSignal<string | null>(null);
const [timeRemaining, setTimeRemaining] = createSignal(45);

// Trigger flash when a new event arrives in draw phase
createEffect(() => {
  const event = props.state.currentEvent;
  const phase = props.state.phase;
  if (phase === 'draw' && event && event.id !== lastEventId()) {
    setLastEventId(event.id);
    setEventFlashActive(true);
    setTimeout(() => setEventFlashActive(false), 2500);
  }
});

// Naive timeRemaining: not a real timer — we just track when phase changes to reset
// Real timer comes from server in Phase 4. For now, reset to 45 on each new planning phase.
createEffect(() => {
  if (props.state.phase === 'planning') {
    setTimeRemaining(45);
  }
});
```
