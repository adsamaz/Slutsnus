- `tradeCardId`: `createSignal<string | null>(null)` — card selected in trade tab
- `tradeTarget`: `createSignal<string | null>(null)` — opponent userId for trade
- `tradeFakeName`: `createSignal<string>('')` — pre-filled with real card name when card selected
- `tradeModalOpen`: `createSignal<boolean>(false)`

**Debuff state:**
- If `props.self.skipNextTurn` is true: render a `<div class="debuff-banner skip">Din tur hoppas över. Du kan inte spela denna runda.</div>` above the tabs. Disable all action buttons (pass `disabled={true}` to buttons).
- If `props.self.highNicEffect` is true: wrap the entire return in `<div class="snusking-hand highnic-blur">`. CSS class `highnic-blur` applies `filter: blur(3px)` (defined in snusking.css). **Play tab:**
- Renders `<SnuskingCard>` for each card in `props.self.hand` with `selected={selectedIds().has(card.instanceId)}`, `onClick={() => toggleCard(card.instanceId)}`, `disabled={props.self.skipNextTurn || props.self.hasCommitted}`
- Actions row below cards:
  - "Spela valda" button — disabled if no cards selected or hasCommitted. onClick: `props.onAction({ type: 'snusking:spend', cardIds: [...selectedIds()] })`
  - "Spela + Öl" button — only shown when `props.self.beer > 0 && selectedIds().size > 0`. onClick: `props.onAction({ type: 'snusking:spend-with-beer', cardIds: [...selectedIds()], beerCardId: 'beer' })`
  - "Aktivera immunitet" button — only shown when `props.self.immunityActive`. onClick: `props.onAction({ type: 'snusking:activate-immunity' })`
  - "Passa" button — onClick: `props.onAction({ type: 'snusking:pass' })`
  - All buttons disabled if `props.self.hasCommitted` or `props.self.skipNextTurn`

**Sabotage tab:**
- Two-step flow:
  1. If `saboTarget()` is null: show all cards in hand. Clicking a card sets `selectedIds` to that card's id. Below cards, show opponent buttons: `<For each={props.opponents}>{(opp) => <button onClick={() => setSaboTarget(opp.userId)}>{opp.username}</button>}</For>`. Disabled if `selectedIds().size === 0`.
  2. If `saboTarget()` is set: show a confirm panel — "Skicka till [target username]?" with two buttons:
     - "Skicka spent snus" → `props.onAction({ type: 'snusking:sabotage-spentsnus', targetPlayerId: saboTarget()!, cardInstanceId: [...selectedIds()][0] })` then reset
     - "Skicka hög-nic" → `props.onAction({ type: 'snusking:sabotage-highnic', targetPlayerId: saboTarget()!, cardInstanceId: [...selectedIds()][0] })` then reset
     - "Avbryt" → reset saboTarget and selectedIds
  - Disabled entirely if `props.self.hasCommitted`

**Trade tab:**
- `<For each={props.self.hand}>{(card) => <SnuskingCard card={card} selected={tradeCardId()===card.instanceId} onClick={() => { setTradeCardId(card.instanceId); setTradeFakeName(card.name); setTradeModalOpen(true); }} />}</For>`
- Trade modal (use `<Modal open={tradeModalOpen()} onClose={() => setTradeModalOpen(false)} title="Erbjud kort">`):
  - Show the card name and points
  - "Visa som:" label + `<input value={tradeFakeName()} onInput={(e) => setTradeFakeName(e.currentTarget.value)} />`
  - "Till:" label + `<select onChange={(e) => setTradeTarget(e.currentTarget.value)}><option value="">Välj spelare</option><For each={props.opponents}>{(opp) => <option value={opp.userId}>{opp.username}</option>}</For></select>`
  - "Skicka" button — disabled if no target. onClick: `props.onAction({ type: 'snusking:trade-offer', targetPlayerId: tradeTarget()!, cardInstanceId: tradeCardId()!, displayName: tradeFakeName() })` then close modal and reset trade signals
